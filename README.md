# Assignment-New-6
Name of the Project : DigiTools

Description
DigiTools is a modern, responsive web application that serves as a one-stop platform for premium digital tools, design assets, templates, and productivity software. Built with a clean and professional UI, the platform is designed to help users discover, explore, and access digital products quickly and efficiently.

Technologies Used

React.js
Vite
Tailwind CSS
JavaScript (ES6+)
GitHub Pages

3 Features:

1. Smart Product Catalog with Search & Filter
The platform features a fully dynamic product catalog powered by a JSON-based data layer. Users can instantly search products by name or description using a live search bar, and filter results by category tags such as Best Seller, Popular, and New. When no results match, a clean empty state appears with a one-click "Clear Filters" option — delivering a smooth and intuitive browsing experience.

2. Real-Time Cart Management with Toast Notifications
Users can add any product to the cart directly from the product card with a single click. Once added, the button state changes to "Added to Cart" with a green confirmation style, preventing duplicate entries. Every cart action — whether adding or removing an item — triggers an instant toast notification, giving users clear and immediate feedback without any page reload.

3. Dynamic Cart Summary with Live Pricing
The cart section provides a detailed order summary showing each product with its icon, name, price, and billing period. The total is calculated automatically as items are added or removed. A prominent "Proceed to Checkout" button displays the live total amount, and upon checkout the cart clears instantly — creating a seamless end-to-end purchase flow entirely on the frontend.
